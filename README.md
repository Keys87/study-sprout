# Documentation

### Development Setup

1. Please ensure you have npm installed and nodejs. To test, please run this command in PowerShell

  ```powershell

  node -v
  ```

2. Make a folder and download the files from the **main** branch

3. Install tailwind CLI by running this command in that folder with the files

  ```powershell
  npm i tailwindcss @tailwindcss/cli
  ```

4. Run tailwindcss through this command

  ```powershell
  npx @tailwindcss/cli -i in.css -o out.css --watch
  ```

  please make sure you are in the same folder as the css files

Additional documentation for tailwind is in **[tailwindcss.com](tailwindcss.com)**

### Design Guide
**Colors**:

1. `#c6ced5` and `#e8e7ea` for **gradients**
2. `#471f0e` for **texts**
3. `#a39f9f` for **shadows**
